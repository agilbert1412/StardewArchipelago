# Stardew Valley Archipelago - Festival Locations

## Festival Locations

This setting allows you to add location checks to various Festival-related events and activities that the player can participate in.

### Option Values

Here are the 3 possible values
- `disabled`
- `easy` **This is the default value, and it is the recommended experience.**
- `hard`

#### Disabled

There are no location checks for participating in festivals. There can still be some festival-related things that you might need to do, to indirectly complete other tasks. For example, you might need to attend the Egg Hunt to purchase strawberry seeds.

But generally, you can skip festivals.

#### Easy

All festivals have one or multiple location checks associated with them. The locations are mostly based on participating, not necessarily performing well.

The location checks can be related to purchasing an item in the festival shop, participating in a festival activity, or sometimes talking to specific people at the festival for some unique dialogue.

#### Hard

All easy festival location checks still exist, but for many of them, the requirements for completion have been increased.

For example, both Easy and Hard have a location check for completing the Grange Display at the [Stardew Valley Fair](https://stardewvalleywiki.com/Stardew_Valley_Fair), but on easy, you get it if you get a score of at least 60, while on Hard, you need a score of at least 90.

Hard also introduces more location checks, so it is also more time consuming to do them all.
